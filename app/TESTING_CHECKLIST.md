# UniLabs Testing Checklist

## Προαπαιτούμενα
1. Εκτέλεση: `cd thesis/unilabs/app && python app.py`
2. Άνοιγμα browser: `http://localhost:5000/login`

## Manual Tests

### 🔐 Authentication
- [ ] Επιλογή "student1" → Redirect στο Dashboard
- [ ] Εμφάνιση username "Test Student" στο sidebar
- [ ] Badge "Φοιτητής" πράσινο
- [ ] "Dev Mode" warning εμφανίζεται
- [ ] Logout → Redirect στο login

### 📊 Dashboard
- [ ] Εμφάνιση ακαδημαϊκού έτους στον τίτλο
- [ ] Κάρτες "Γρήγορη Εγγραφή" και "Οι Εγγραφές μου" (μόνο για students)

- [ ] Sidebar toggle λειτουργεί

### 📝 Cascading Dropdowns (Tab: Εγγραφή σε Εργαστήριο)
- [ ] Εξάμηνο dropdown φορτώνει επιλογές
- [ ] Επιλογή εξαμήνου → Ενεργοποίηση dropdown Μαθήματος
- [ ] Επιλογή μαθήματος → Ενεργοποίηση dropdown Εργαστηρίου
- [ ] Επιλογή εργαστηρίου:
  - [ ] Εμφάνιση πληροφοριών εργαστηρίου (όνομα, λήξη εγγραφών)
  - [ ] Έλεγχος κατάστασης εγγραφής
  - [ ] Ενεργοποίηση dropdown Τμήματος
- [ ] Επιλογή τμήματος:
  - [ ] Progress bar πληρότητας
  - [ ] Στοιχεία υπεύθυνου καθηγητή (δεξιά κάρτα)
  - [ ] Ενεργοποίηση κουμπιού "Εγγραφή"

### ✅ Registration
- [ ] Κλικ "Εγγραφή" → Spinner loading

### 🔄 Change Group
- [ ] Επιλογή ίδιου εργαστηρίου μετά την εγγραφή
